## Description

In this project, we are using [Nest](https://github.com/nestjs/nest) framework .

## Installation

```bash
$ npm install
```

## Running the app us

```bash

# simple way using docker-compose
$  make up
```

or

```bash
# the hardway :D
# launch database with docker
$ npm run start:db

# launch serveur
$ npm run start:dev
```

## Api documentation

go to swagger => http://localhost:5000
